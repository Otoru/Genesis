{{< callout type="info" >}}
Genesis requires Python 3.10 or higher.
{{< /callout >}}

{{% steps %}}

### Install Genesis

Install Genesis in your environment using `pip`:

```bash
pip install genesis
```

### Verify Installation

Verify that Genesis is installed correctly:

```bash
genesis --version
```

### Next Steps

With Genesis installed, explore the available guides in the [homepage](../) to learn how to use it.
